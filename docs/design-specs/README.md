# LIT Design Specs

Primary design source:
docs/design-specs/lit-design-system/

Claude Code must inspect this bundle before editing production code.

Use the bundle as the visual and component reference for:
- Dashboard
- Command Center
- Company Detail
- Outbound / Outreach
- Pulse
- Affiliate
- Shared sidebar, globe, colors, and typography

Do not loosely interpret the designs.
Do not paste prototype code directly into production.
Convert the designs into proper app components while preserving real data, auth, routes, Supabase, and Stripe contracts.

Pages not fully covered by this bundle:
- Settings
- Billing
- Search / Discover
- Website / Blog
- RFP Studio
- Widgets
- Admin Dashboard

For those pages, use separate specs or existing app requirements.
