- **Border**: 2px solid blue-200
- **Icon**: Sparkles (âœ¨) in blue-600
- **Text**:
  - Heading: bold, blue-900
  - Body: smaller, blue-700
- **Close Button**: Blue-600 with hover effect

### Sample Badges
- **Position**: Absolute top-right (-2px offset)
- **Style**: Small rounded pill
- **Colors**: Blue-600 background, white text
- **Size**: 10px font, bold
- **Shadow**: Small drop shadow for depth

### Card Animations
- **On Load**: Staggered fade-in (100ms delay per card)
- **On Hover**: Subtle scale (1.02) + shadow increase
- **On Click**: Scale down (0.98) feedback

### Company Cards
- **Header**: Company name (truncated if long)
- **Subheader**: City, State
- **Metrics**: Icons + formatted numbers
- **Routes**: Flag/pin icon + route summary
- **Actions**: View Details + Save buttons
- **State**: Visual indicator when saved (filled heart)
